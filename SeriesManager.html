Datenschutzerklärung für die App Series Manager

Einleitung

Der Schutz Ihrer persönlichen Daten ist uns ein wichtiges Anliegen. Diese Erklärung informiert darüber, welche Daten in der App Series Manager erfasst und wie diese verarbeitet werden.

Verantwortlicher

Thomas Herzog
(cendia.de)
E-Mail: info@cendia.de

Datenverarbeitung

Die App dient der Verwaltung von Serienfolgen, Watchlisten und Nutzerpräferenzen. Alle eingegebenen Daten werden ausschließlich lokal auf dem Gerät gespeichert. Es erfolgt keine automatische Übertragung an Server oder Dritte.

Erfasste Daten

  • Seriennamen, Staffel- und Episodeninformationen
  • Watchlist-Einträge, Bewertungsnotizen, Status
  • Individuelle Einstellungen der App

iCloud-Speicherung (optional)

Auf ausdrücklichen Wunsch des Nutzers können Daten in der iCloud gespeichert werden (z. B. zur Synchronisierung zwischen Geräten). Die Aktivierung erfolgt manuell durch den Nutzer. Es werden nur Daten gespeichert, die direkt in der App erzeugt wurden.

Datensicherheit

Alle Daten bleiben auf dem Gerät oder – bei Nutzung der iCloud-Funktion – im geschützten iCloud-Bereich des Nutzers. Es erfolgt keine Weitergabe an Dritte.

Rechte des Nutzers

Da keine personenbezogenen Daten verarbeitet werden, gelten die klassischen Datenschutzrechte nur eingeschränkt. Bei Fragen zur Datennutzung steht der Entwickler gerne zur Verfügung.

Änderungen

Diese Erklärung kann bei Bedarf angepasst werden. Die jeweils aktuelle Version ist in der App abrufbar.

Privacy Policy for the Series Manager App

Introduction

Your privacy is important to us. This policy explains what data is collected in the Series Manager app and how it is handled.

Responsible Party

Thomas Herzog
(cendia.de)
Email: info@cendia.de

Data Processing

The app is used to manage TV series episodes, watchlists, and user preferences. All entered data is stored locally on the device. There is no automatic transmission to any server or third parties.

Collected Data

  • Series names, season and episode information
  • Watchlist entries, rating notes, viewing status
  • User-defined app settings

iCloud Storage (optional)

Upon explicit request by the user, data can be stored in iCloud (e.g. to sync across devices). This function must be manually enabled by the user. Only data created within the app is stored.

Data Security

All data remains on the device or – when using iCloud – within the user’s private iCloud space. No data is shared with third parties.

User Rights

Since no personal data is processed, typical data protection rights apply only in a limited form. For any questions, the developer is available via the contact above.

Changes

This policy may be updated when necessary. The current version is available within the app.
